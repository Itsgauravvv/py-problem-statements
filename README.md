# Python Healthcare Problem Statements

This repository contains Python programs that solve healthcare-related problem statements using conditional statements, user input, and decision-making logic.

## 📂 Files

### 1. HealthPatient.py
A Python program that evaluates a patient's health condition based on:
- Body Temperature
- Heart Rate
- Oxygen Saturation (SpO₂)

**Features:**
- Determines patient risk level (Low, Medium, High, Critical)
- Provides medical recommendations
- Gives priority consultation for senior citizens

---

### 2. HospitalHealth.py
A Python program that determines insurance approval and treatment eligibility.

**Features:**
- Validates insurance status
- Checks policy validity
- Decides cashless treatment eligibility
- Handles pending approvals and rejections
- Applies senior citizen benefits
- Handles emergency case notifications

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Itsgauravvv/py-problem-statements.git
   ```

2. Navigate to the project folder:
   ```bash
   cd py-problem-statements
   ```

3. Run any program:
   ```bash
   python HealthPatient.py
   ```
   or
   ```bash
   python HospitalHealth.py
   ```
